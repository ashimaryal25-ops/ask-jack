# Ender 3 V3 KE — Complete Beginner Workflow
machine: Ender 3 V3 KE
category: 3D printing — full workflow

## "I want to 3D print something. Where do I start?"

Here is the complete process from idea to finished object.

---

## Step 1 — Get a model file

You need a .stl or .obj file of the object you want to print.

**If someone already made it (easiest):**
Search for it on Thingiverse (www.thingiverse.com) or Printables (www.printables.com). Download the .stl file.

**If you need something custom:**
Design it in Tinkercad (www.tinkercad.com) — free, browser-based, no install needed. Export as .stl when done.

---

## Step 2 — Slice the model on a computer

1. Open **Ultimaker Cura** on a computer (download free at www.ultimaker.com/software)
2. First-time only: Add printer → Non-networked → Creality → Ender 3 V3 KE
3. Open your .stl file in Cura
4. Check settings: Material = PLA, Layer height = 0.2mm, Infill = 20%
5. Enable Supports if your model has overhangs
6. Insert a USB drive into the computer
7. Click **Slice** → **Save to Disk** → save to USB drive
8. Eject USB

---

## Step 3 — Load filament (if needed)

If the printer already has PLA loaded in the color you want, skip to Step 4.

To change filament:
1. Turn on printer (red switch, left side)
2. Tap screen → Prepare icon → Extrude/Retract tab
3. Tap Retract → pull out old filament → wind it back on spool
4. Thread new filament through detector and into extruder
5. Tap Extrude → wait until new color flows consistently

---

## Step 4 — Print

1. Put on safety glasses
2. Clear the print bed — make sure it is clean and flat
3. Insert USB into the side of the control panel
4. Turn on printer (red switch) → tap screen to wake
5. Tap File icon → USB Drive tab → select your .gcode file
6. Review details → tap **Print**
7. Printer auto-levels, heats up, and starts printing
8. Watch the first 2–3 minutes — first layer should stick flat and smooth
9. Write your name on the dry erase board
10. Come back when done (printer stops automatically)

---

## Step 5 — Remove your print

1. Wait for bed to cool
2. Lift the flexible magnetic bed plate off
3. Flex it gently — print pops off
4. Replace bed plate flat on printer
5. Clean up scraps, erase your name from the board

---

## That's it. You made a 3D print.

**Something went wrong?**
- Print not sticking → clean bed with isopropyl alcohol and try again
- Filament not coming out → nozzle clog, ask ICL staff
- Grinding noise → stop print immediately, ask ICL staff
