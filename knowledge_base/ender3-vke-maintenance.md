# Ender 3 V3 KE — Maintenance & Troubleshooting
machine: Ender 3 V3 KE
category: 3D printing — maintenance

## Bed plate care
- Always wait for the bed to cool before removing prints
- To remove a print: lift the magnetic bed plate off, then flex it gently with both hands — the print pops off cleanly
- If filament scraps are stuck to the bed: scrape lightly with a blade, then wipe clean
- If the first layer keeps not sticking: apply a thin, even layer of **glue stick (solid adhesive)** to the bed surface before printing
- The bed plate is a consumable — replace it when it no longer holds prints well

## Lubrication (periodic)
The screw rods, optical shafts, and guide rails need occasional lubrication to keep movement smooth.
- Apply lubricant to: screw rod area, optical axis area, guide rail area
- Ask ICL staff if you are unsure what lubricant to use or how often

## Nozzle replacement (staff only)
If the nozzle is clogged and cleaning does not fix it, it may need replacement. This is a staff task — do not attempt it yourself. For reference:
- Nozzle must be preheated before removal
- A tool must hold the heating block while the nozzle is unscrewed to avoid damage
- Standard nozzle diameter is 0.4mm

## Cleaning
- Clean the printer body with a dry cloth while powered OFF
- Wipe dust, filament scraps, and debris off guide rails regularly
- Do not use wet cloths on electronics

## Power and voltage
- Check the voltage switch on the power supply before first use
- 100–120V mains: set switch to 115V
- 200–240V mains: set switch to 230V (default)
- Using the wrong voltage setting can damage the printer

## Slicing software options
The Ender 3 V3 KE is compatible with:
- **Ultimaker Cura 5+** (recommended at ICL)
- **Creality Print** (official Creality slicer, free at crealitycloud.com)
- Simplify3D

Supported file formats for slicing input: STL, OBJ, 3MF, AMF
